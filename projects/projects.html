MasterMind Gems

A reimagining of the classic board game MasterMind.

Technologies

HTML5
CSS3
JavaScript
Heroku

Details

Web-based gaming project. The aim of this project was to learn more about the HTML5 canvas property with regards to drawing multiple dynamic graphics.
Staying true to the original game, this web-based version is optimized to fit any screen on initial startup, giving the player a view of the entire board at once.

Fringe Memory Game

A mini-tribute to a beloved sci-fi tv show, Fringe(2008 - 2013).

Technologies

Python
Pygame

Details

This is a desktop memory game in python.
One of the things I loved about Fringe (other than fringe science) was the weird hidden glyphs and easter eggs scattered throughout the show. Subtle differences in each glyph gave a different meaning.
The aim of this project was to learn more about using pygame. Making a memory game using glyphs from a favourite show was an easy choice.
