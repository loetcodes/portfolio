Map Imagery Slider

Web app used to compare multiple imagery on a map.

Technologies

ArcGIS Web API
JavaScript
HTML
CSS
Heroku

Details

This is an app currently in use by a client company for creating a time-based imagery viewer.
The main purpose of the app is to compare imagery on an interactive map. It features a list of imagery available for comparison, and a slider that swipes over the imagery for comparison. The user is able to select the imagery for a set of given control buttons.
Below is a public demo that is a recreation of the app using freely available imagery hosted on Heroku.

Reach out if you want to view a sample of a client's app that is publicly available.

Fringe Memory Game

A mini-tribute to a beloved sci-fi tv show, Fringe(2008 - 2013).

Technologies

Python
Pygame

Details

This is a desktop memory game in python.
One of the things I loved about Fringe (other than fringe science) was the weird hidden glyphs and easter eggs scattered throughout the show. Subtle differences in each glyph gave a different meaning.
The aim of this project was to learn more about using pygame. Making a memory game using glyphs from a favourite show was an easy choice.

DecodeSpace

Blogging app used to document creative explorations

Technologies

Python
Django
Postgresql
HTML5
CSS3
JavaScript
Heroku

Details

Blogging application built on python and the django framework in the backend.
It implements the model-view-control pattern using django's base setup in its application design. Python is used to create, read, update and delete posts in the database.
This was a learning opportunity to deep-dive into building web applications using python and django, as well as working with postgresql databases.
